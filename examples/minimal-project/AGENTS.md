# Minimal Project AGENTS

This project uses the `opencode-agent-pack` default single-entry workflow.

## How to Use
- Talk to the default entry `orchestrator`
- Use `/task <requirement>` or natural language

## Behavior
- orchestrator auto-runs triage
- system auto-selects lane + tiers
- subagents are auto-dispatched
- escalation is automatic when needed
- final closure is always by orchestrator
